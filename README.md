# React-introduction

Several exercises to practise ReactJS.

# Technologies

- ReactJS
- JSX
- HTML
- CSS

# How to test

1. Go to current folder of project in terminal
2. Run "npm install"
3. Go to folder public
4. Run "npm start"
5. The project will be launch directly in navigator
6. Here are the different routes available, each routes correspond to an exercise :

   - http://localhost:3000/ ==> form exercise
   - http://localhost:3000/movies ==> movies exercise
   - http://localhost:3000/person ==> person exercise
   - http://localhost:3000/rating ==> rating exercise
   - http://localhost:3000/counter/2 ==> counter exercise

7. Enjoy !
